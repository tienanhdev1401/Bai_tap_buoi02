# User Management System - MongoDB

## Hướng Dẫn Chạy Project

### 1. Cài đặt dependencies
```bash
npm install
```

### 2. Tạo dữ liệu mẫu
```bash
npm run migrate
```

### 3. Chạy server
```bash
npm run dev
```

### 4. Truy cập ứng dụng
- Trang chủ: http://localhost:3000/home
- Danh sách users: http://localhost:3000/get-crud
- Thêm user: http://localhost:3000/crud

**Công nghệ:** Node.js, Express, MongoDB, Mongoose, EJS, Babel, bcryptjs
